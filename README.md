# 🔐 Modern Login Page - React.js

Este projeto é uma interface de login de alto nível desenvolvida com **React.js**, focada em aplicar conceitos modernos de UI/UX como o **Glassmorphism**. O objetivo principal foi criar uma experiência de usuário fluida, com um layout "premium" e código bem estruturado.

---

## 🛠️ Tecnologias Utilizadas

* **React.js**: Biblioteca principal para construção da interface.
* **JavaScript (ES6+)**: Lógica de componentes e estados.
* **CSS3 Avançado**: 
    * **Glassmorphism**: Efeito de vidro fosco com `backdrop-filter`.
    * **Radial Gradients**: Fundo profundo e moderno.
    * **Flexbox**: Alinhamento e responsividade.
* **Git & GitHub**: Versionamento de código.

---

## 🚀 Diferenciais do Projeto

* **Design Glassmorphism**: Uso de transparências e desfoque de fundo para uma estética moderna.
* **UI/UX Refinado**: Ajustes precisos de padding, margin e tipografia para maximizar a legibilidade.
* **Código Limpo**: Componentização eficiente seguindo padrões de mercado.
* **Responsividade**: Totalmente adaptável para diferentes tamanhos de tela.
